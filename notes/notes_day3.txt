dynamic programming language

data types
operators
expressions/control flow statements

data types:
	- number
	- boolean
	- string
	- array
	- object

	- function
	- undefined

operators:
	arithmatic: +, -, *, /, %, ++, --
	conditional: >, <, >=, <=, &&, ||, !, ?:
	bitwise: &, |, ^, <<, >>
	equality: ==, !=

control flow statements:
	conditional statements:
	 - if...else, if, nested if...else, switch..case
	Looping:
		while, do..while
	iterational:
		for, for..in, for..each
	jump statements:
		break, continue etc.
