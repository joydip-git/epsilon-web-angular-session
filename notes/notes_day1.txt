C, C++, C#, Java

HTML: (Hypertext Markup Language)
	- standard markup language for creating web pages
	- describes the structure of the web page
	- it does this by using tags (elements)
	- HTML tags tells browser how to display the content

HTML document:
	- head: contains some information about HTML document
	- body: contains the actual design/structure of the document

Tags/Elements:
	a tag/element can contain another tag/element or a text as child
	a tag/element can be empty: empty tag

Lists:
	a. unordered list (ul)
	b. ordered list (ol)
		attribute: type
		possible values: 1 (default), A, a, i, I
		attribute: start
		possible values: anything (only numeric)
	** both OL and UL lists contain list items (li)

	c. description list: (dl)
		item (term): dt
		desciption: dd

Meta tag:
	author, refresh, keywords, charset, viewport etc.
style:

	style-atribute-name:value
	example:
		background-color:blue
		font-size:'100px'
		border-radius:5px

1. inline (same HTML file on the element itself)
2. outline (same HTML file outside the element and link with the element in 3 different ways)
	use selectors to link the style with an element:
	a. element selector
	b. id selector
	c. class selector
3. outfile (outside HTML file - separate *.css file and link it with HTML by link tag)	

script, link tag
